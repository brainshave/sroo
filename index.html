What it is actually?

Slideshow Bob takes simple well-formatted HTML document and turns it to nice navigable slides.

How to navigate through slides?

Navigate through slides using arrows: up/down between slides and left/right between subslides.

How to prepare slides?

Each slide is a <section> inside <body>

Each slide consist of (optional) <h1> header. Any nodes that will follow will be presented as sub-slides.
